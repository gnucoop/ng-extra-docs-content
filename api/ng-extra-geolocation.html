API reference for Ng-Extra geolocation

import {GeolocationModule} from '@gnucoop/ng-extra/geolocation';

Services

Geolocation

Service to get or watch current position (geolocation)

Properties
NameDescription

geolocationAvailable: boolean

Whether the HTML5 Geolocation API is available

Methods
getCurrentPosition

Gets a stream that emits when the current position is acquired via the HTML5 Geolocation API

Parameters

options?

PositionOptions

Option properties to pass as a parameter of Geolocation.getCurrentPosition()

Returns
Observable<Position>

A stream of the acquired position

watchPosition

Gets a stream that emits when the position acquired via the HTML5 Geolocation API at intervals of intervalMs milliseconds

Parameters

options?

PositionOptions

Option properties to pass as a parameter of Geolocation.getCurrentPosition()

intervalMs

number = 1000

Milliseconds between each position acquisition

Returns
Observable<Position>

A stream of the acquired position
